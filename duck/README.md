# Ducker

It's can help you to debug.
Say something, and answers you.

**Example**:

| You              | Duck     |
|:-----------------|:---------|
| Hi.              | Quack?   |
| Can you help me? | Quack!   |
| I must debug.    | Quack... |
| He say something | Quack?   |
| 'segment fault'. | Quaaack. |
| Because I....... | Quack!   |
| Oh I found !     | Quack.   |
| Thanks!          | Quack!   |

He just speak. It's just a duck.
Or a JavaScript code.
